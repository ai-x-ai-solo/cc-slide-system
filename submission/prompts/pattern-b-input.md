# Pattern B 入力例：法人・業務改善訴求型

---

## 入力パラメータ

```yaml
service_name: "本質のClaude Code 完全攻略 7dayブートキャンプ"

target: |
  中小企業の経営者・役員
  管理職・業務改善担当者
  社内のDX推進・AI導入を検討している人
  「AIを入れたいが何から始めればいいか分からない」組織

main_appeal: |
  社内の繰り返し作業をAIで自動化できる
  外注費・人件費を削減しながら生産性を上げられる
  専門家に頼まずに自社でAIツールを構築できる

emotion: |
  「これなら自社の業務に使えそう」
  「うちの社員でも7日で使えるようになるなら安い投資だ」
  「外注に頼らず内製化できる」

design_tone: |
  白×青
  信頼感・誠実さ・整理された印象
  図解・比較表を多用
  数字はデータとして見せる（グラフ的に）

priority_info: |
  - 月間300本の講義動画を自動制作という実績
  - 法人AIコンサル：月額50万〜200万円という市場価格比較
  - 社内12種のシステムがそのまま手に入るという実用性
  - 「100個以上のシステムを構築」という再現性の証明

remove_info: |
  - 副業・案件獲得・収入アップのメッセージ
  - 「早い者勝ち」「0.1%」という煽り系表現
  - 個人の成功体験（法人判断には不要）

available_images:
  - "assets/references/profiles/mikami-profile.png"
  - "assets/references/profiles/sato-profile.png"

fallback_if_no_image: |
  みかみ：「創業5年・年商30億・AI月商5億」の実績カードで代替
  佐藤：「社内12種のシステム構築実績」のテキストリストで代替
```

---

## 期待するスライドの雰囲気

- カバー：「AI導入に、外注は必要ない。」
- 問題提起：「社内の繰り返し作業に、毎月いくらかかっていますか？」
- 解決策：「Claude Codeは、日本語で動かせるAI開発ツールです。」
- 実績：「月間300本の動画を、4人で自動制作しています。」
- 価格：「法人AIコンサルの相場は月50〜200万円。このブートキャンプは198,000円。」
