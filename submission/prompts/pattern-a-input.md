# Pattern A 入力例：副業・案件獲得訴求型

---

## 入力パラメータ

```yaml
service_name: "本質のClaude Code 完全攻略 7dayブートキャンプ"

target: |
  AI副業に興味がある個人
  スキルプラス受講生
  月10万円以上の案件を獲得したい人
  副業収入を作りたいが何から始めればいいか分からない人

main_appeal: |
  月10万円の案件を7日で取れるスキルが身につく
  Claude Codeを使いこなしている日本人は推定0.1%未満
  早い者勝ち・先行者利益

emotion: |
  「今動けば先行者になれる」
  「このまま動かないと置いていかれる」
  「7日で人生が変わるかもしれない」

design_tone: |
  黒×パープル
  未来感・緊迫感・高級感
  数字を大きく・太く・目立たせる
  余白を意識的に使う

priority_info: |
  - 月10万円案件という具体的な数字
  - 7日間という短期間
  - 0.1%という希少性
  - 佐藤さんの「元未経験→初月1,000万円案件」のストーリー

remove_info: |
  - 法人向けの業務効率化・自動化の訴求
  - 専門用語（API・自動化・Python など）
  - 特典の詳細一覧（合計金額だけで十分）

available_images:
  - "assets/references/profiles/mikami-profile.png"
  - "assets/references/profiles/sato-profile.png"

fallback_if_no_image: |
  みかみ：「月商5億 / 受講生6,000名」のテキストカードで代替
  佐藤：「元公務員 → 初月1,000万円案件」のストーリーラインで代替
```

---

## 期待するスライドの雰囲気

- カバー：「7日で、月10万円の案件を取る。」という1行が画面を支配する
- 危機感スライド：「AIを使える人と使えない人の差は、今この瞬間も広がっている」
- 価格スライド：「1日あたり251円。月々7,532円〜。」
- CTA：「先に動いた人だけが、圧倒的に有利になる。」
