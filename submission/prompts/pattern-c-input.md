# Pattern C 入力例：初心者安心訴求型

---

## 入力パラメータ

```yaml
service_name: "本質のClaude Code 完全攻略 7dayブートキャンプ"

target: |
  AIに興味があるがどこから始めればいいか分からない人
  PCは使えるが「プログラミング」と聞くと怖い人
  「自分には無理かも」と思っている人
  副業に興味があるが自信が持てない人

main_appeal: |
  プログラミング不要・日本語で話しかけるだけで動く
  質問し放題・補講あり・一人にしない
  1期生は未経験から7日でシステムを完成させた

emotion: |
  「自分でもできそう」
  「一人じゃないから安心」
  「まずやってみよう」

design_tone: |
  明るい・温かい・柔らかい
  余白を多く取る・詰め込まない
  アクセントカラーは明るいグリーン or コーラル
  数字は「7日」「1日30分でOK」など親しみやすい形で

priority_info: |
  - 「プログラミング不要・日本語でOK」というハードル除去
  - 3段階サポート（予習・ブートキャンプ・補講）
  - 「1期生が未経験から7日でシステム完成」という実績
  - 1日あたり251円という払えそう感
  - 「PCで何かしらの作業をしている人なら誰でも」という入口の広さ

remove_info: |
  - 「月商5億」「1,000万円案件」など大きすぎる数字
  - 法人向け・業務改善の訴求
  - Claude APIなど技術用語
  - 「早い者勝ち」「AI格差」など煽り表現

available_images:
  - "assets/references/profiles/mikami-profile.png"
  - "assets/references/profiles/sato-profile.png"

fallback_if_no_image: |
  みかみ：「受講生6,000名以上が信頼する講師」のシンプルテキストで代替
  佐藤：「プログラミング未経験から100以上のシステムを作った人が教える」で代替
```

## 期待するスライドの雰囲気

- カバー：「プログラミングは、いりません。」
- 問題提起：「『毎回同じ作業してるな』と思ったことはありますか？」
- 解決策：「Claude Codeは、日本語で話しかけるだけで動きます。」
- サポート訴求：「予習・本番・補講の3段階。迷ったらすぐ質問できます。」
- 実績：「1期生は、未経験から7日でシステムを完成させました。」
- 価格：「1日あたり251円。コンビニのランチ代より安い。」
- CTA：「まずやってみることが、一番の近道です。」
