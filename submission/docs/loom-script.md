## 0:00〜0:20 挨拶と概要

「はじめまして、荒井昇と申します。
今回の採用評価タスクとして、Claude Code ブートキャンプの販売スライド自動生成システムを作りました。

このシステムは、サービス情報をYAMLファイルに入力するだけで、ターゲット別の販売スライドをPowerPointで自動出力できます。
今日は3分でその全体像をご説明します。」

---

## 0:20〜0:50 システム概要

「まず、システムの全体構成です。

入力はservice_info.yamlというファイル一つ。
ここにサービス名・価格・講師情報・特典・Q&Aをすべて記載しています。

これを読み込んで、Claude APIがスライドのコピーと構成をJSON形式で生成し、
python-pptxがそれをPowerPointファイルとして出力します。

コマンド一つで実行でき、3パターン分のスライドが数十秒で生成されます。」

（画面：ターミナルで python3 generate_v6.py を実行するところを見せる）

## 0:50〜1:30 3パターンの説明

「今回は3つのターゲット訴求パターンを作りました。

Pattern A は副業・案件獲得訴求型。
『月10万円の案件』『Claude Codeを使いこなしている日本人は0.1%未満』という数字を主役に、
今動けば先行者になれるという感情を引き出す構成です。
デザインは黒×パープルで未来感・緊迫感を出しています。

Pattern B は法人・業務改善訴求型。
経営者・管理職向けに、業務効率化・自動化・生産性向上を訴求します。
白×青の信頼感あるデザインで、図解とデータを使った構成にしています。

Pattern C は初心者安心訴求型。
PCが苦手な方・未経験者向けに、『日本語でOK』『質問し放題』を前面に出し、
明るく柔らかいデザインで安心感を演出しています。」

（画面：3パターンのスライド1枚目を並べて見せる）

## 2:00〜2:25 AIワークフロー

「AI活用の役割分担をご説明します。

Claude Codeは、ファイル設計・コーディング・システム全体の実装に使いました。
Claude APIは、スライドのコピー・構成をJSON形式で生成するのに使いました。
ChatGPTは、『何を削るか』『感情導線をどう設計するか』という上流の設計相談に使いました。

最初はデザイナーエージェントで自動生成しましたが、出来上がりを見て違和感があり、
ChatGPTと壁打ちして構成から見直しました。
『読む資料ではなく、感じさせる資料』という軸が見えてから、一気に整理できました。」

## 2:25〜2:50 こだわった点・削った点

「一番こだわったのは『削る勇気』です。

特典12種・講義13テーマ・講師の詳細経歴、全部載せたくなりますが、
全部見せると何も伝わりません。
役割ごとに情報を圧縮して、1枚で1つの感情だけを残しました。

また、画像の差し替えが簡単にできる設計にしました。
assets/references/フォルダに指定のファイル名で画像を置くだけで、
次回以降の生成に自動で反映されます。
コードを触らずに運用できる設計です。」
